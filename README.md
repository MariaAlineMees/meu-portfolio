# Documentação do Projeto: Portfólio Profissional - Maria Aline Mees

## 1. Introdução
Este projeto é uma landing page desenvolvida como Atividade Prática Extra para o curso de Introdução à Programação Front-End e Back-End. O objetivo é consolidar minha transição de carreira para a tecnologia, unindo **12 anos de experiência profissional** em gestão, logística e ensino com o inicio da graduação em **Sistemas de Informação na UFBRA**. A página apresenta meu percurso técnico em um ambiente responsivo, demonstrando competências em backend e arquitetura de sistemas.

## 2. Tecnologias e Arquitetura Web
O projeto aplica conceitos modernos de desenvolvimento para garantir performance e acessibilidade:
* **HTML5:** Estrutura semântica organizada em seções estratégicas: Início (Hero), Sobre Mim, Projetos, Habilidades e Contato.
* **CSS3:** Estilização Dark Mode com variáveis personalizadas. Utiliza **Flexbox** para alinhamento e **Grid Layout** para a organização responsiva dos cards de habilidades e projetos.
* **JavaScript:** Implementação de lógica para manipulação da área de transferência via **Clipboard API**, oferecendo feedback visual instantâneo ao usuário.

## 3. Portfólio de Projetos e Desafios Técnicos
A documentação destaca a evolução técnica através de projetos reais:

1. **CapivaraIA - Assistente Virtual(SEDEAD):** Solução de IA para desburocratização governamental feito durante a Hackweek em 2025. Fui a responsável pelo desenvolvimento integral do microsserviço backend em **Java**. O maior desafio foi garantir a escalabilidade através de containers na **AWS** e a automação via **Terraform** e **GitHub Actions**.
2. **Undead Rising:** RPG de sobrevivência em **Java Puro (JDK 11)**. Focado na aplicação rigorosa de **POO** e padrões de projeto para gerenciar estados complexos de jogo e lógica de combate por turnos.
3. **Meu Roteiro:** Sistema Fullstack distribuído com **Spring Boot 3** e **Angular**. O diferencial técnico reside no uso de **RabbitMQ** para mensageria assíncrona e integração com **Spring AI** para geração de conteúdo dinâmico.

## 4. Diferenciais Profissionais (Soft Skills)
Minha transição de carreira aos 33 anos traz uma bagagem de **12 anos em gestão e comunicação**, o que facilita o trabalho em equipes ágeis (Scrum), o Pair Programming e a visão de software como solução para dores reais de negócio.

## 5. Instruções de Execução
Para testar a aplicação localmente:
1. Extraia o arquivo compactado (.zip) enviado no portal.
2. Certifique-se de que `index.html`, `style.css` e `script.js` estejam no mesmo diretório.
3. Abra o `index.html` em qualquer navegador moderno (Chrome, Edge ou Firefox).
