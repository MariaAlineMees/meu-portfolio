Sobre mim

Olá, eu sou a Maria Aline Mees! Aos 33 anos, estou vivendo minha transição de carreira para a tecnologia, unindo 12 anos de experiência profissional como Personal Trainer, Gestora e Professora com a graduação em Sistemas de Informação. Minha trajetória anterior me deu uma base sólida em gestão, logística e finanças, o que me permite olhar para o desenvolvimento de software não apenas como um desafio técnico, mas como uma ferramenta para resolver problemas de negócio e fidelizar clientes.

Atualmente, sou uma entusiasta do Backend. Através de formações intensivas como o +DEVS2BLU, Entra21 e Bootcamp Santander, desenvolvi projetos práticos utilizando Java, Spring Boot e Angular para o desenvolvimento de aplicações web modernas.

Meu foco técnico envolve Modelagem SQL, RabbitMQ e aplicação rigorosa dos princípios de Clean Code e POO. Em 2024, realizei um intercâmbio de imersão em Inglês, focando em fluência para ambientes de trabalho globais. Minha experiência com ensino me torna uma profissional preparada para colaboração em equipes.

Meus Projetos

CapivaraIA - Assistente Virtual (SEDEAD)

A CapivaraIA é uma solução de IA desenvolvida para a Secretaria de Estado da Administração (SEDEAD) durante um hackweek em 2025. Como responsável pelo desenvolvimento integral do microsserviço [service-java], implementei uma API REST robusta integrada a um ecossistema de 5 repositórios.

Arquitetura & Cloud: Implementação em containers via Amazon na AWS, com tráfego gerenciado por Load Balancer e isolamento em rede privada. Deploy 100% automatizado através de pipelines de CI/CD via GitHub Actions e infraestrutura como código com Terraform.

Tecnologias: Java, Spring Boot, AWS, Docker, Terraform e GitHub Actions.

Undead Rising - RPG (Java Puro)

O "Undead Rising" é um simulador de apocalipse zumbi desenvolvido durante o curso +Devs2Blu. O projeto prioriza a aplicação rigorosa de Programação Orientada a Objetos (POO) para gerenciar um sistema de combate por turnos, inventário dinâmico e classes de personagens.

Destaques Técnicos: Utilização do Apache Maven para automação de build e gerenciamento de dependências, com interface gráfica desenvolvida em Java Swing/AWT.

Tecnologias: Java 11, Maven, Java Swing/AWT e POO.

Meu Roteiro - Gestão de Viagens Fullstack

Projeto final do +Devs2Blu que utiliza uma arquitetura distribuída para gerenciar roteiros de viagem. A aplicação conta com 4 serviços independentes e integra Inteligência Artificial via Spring AI e Ollama para sugestões inteligentes.

Backend & Segurança: Implementação de mensageria assíncrona com RabbitMQ para notificações e segurança robusta com Spring Security e criptografia BCrypt. Todo o ambiente é orquestrado via Docker Compose.

Tecnologias: Java 17, Spring Boot 3, Angular, RabbitMQ, MySQL, Docker e Spring AI.
